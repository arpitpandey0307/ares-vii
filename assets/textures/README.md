# Textures Directory

Place texture files here:

- `earth_map.jpg` - Earth surface texture
- `earth_normal.jpg` - Earth normal map
- `earth_clouds.jpg` - Earth cloud layer texture
- `mars_map.jpg` - Mars surface texture
- `mars_normal.jpg` - Mars normal map
- `mars_displacement.jpg` - Mars terrain displacement map

You can download free planet textures from:
- https://www.solarsystemscope.com/textures/
- https://planetpixelemporium.com/planets.html
